Усовершенствованные функции переводов

Платформа EDC Blockchain предоставляет пользователям расширенный функционал для проведения транзакций.

Название блока о котором идет речь

Код-пакет

Дополнительный уровень защиты для ваших переводов — при создании “код-пакета” вы генерируете секретный код. Только обладатель кода сможет воспользоваться вашими средствами.

Красный конверт

Расширенная версия “код-пакета” — средства из красного конверта могут быть распределены между несколькими получателями. Эта функция отлично подходит для проведения акций лояльности, маркетинговых и скидочных программ для ваших клиентов.

Анонимные транзакции

Полностью анонимные переводы, отправитель которых не может быть определен. Инструмент, который позволяет вам всегда оставаться анонимным, когда это действительно необходимо!
